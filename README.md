# Azuki OS

Azuki OS は、ブラウザ上で動作する擬似OS環境です。
HTML / CSS / JavaScript のみで実装されています。

## 起動方法
 https://yuzuchan2525.github.io/Azuki_os/

## 注意
- 実際のOSではありません
- データはブラウザの localStorage に保存されます
- 不具合が出た場合は初期化してください

## 動作環境
- PCブラウザ（chrome、bingは動作確認済み）
- スマホは非対応でスタートメニューが隠れたりしますが一応操作はできます
## OSバージョン
- 現在のバージョンは2.8です。
- 表示バージョンが古い場合は Ctrl+F5 を試してください
## BIOSについて
- BIOS画面でF2を押すとFPS表示など便利機能を設定できます
