# Azuki OS

Azuki OS は、ブラウザ上で動作する擬似OS環境です。
HTML / CSS / JavaScript のみで実装されています。

## 起動方法
- BIOS画面：
  https://yuzuchan2525.github.io/azuki-os/bios.html
- 直接起動(不具合が起きる可能性があります)：
  https://yuzuchan2525.github.io/azuki-os/

## 注意
- 実際のOSではありません
- データはブラウザの localStorage に保存されます
- 不具合が出た場合は初期化してください

## 動作環境
- PCブラウザ（chrome bingは動作確認済み）
- スマホは非対応でスタートメニューが隠れたりしますが一応操作はできます
